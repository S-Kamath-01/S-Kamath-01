# 🚀 Featured Projects

## 💳 Fraud Detection (Cost-Optimal)
End-to-end fraud detection system built on the PaySim transaction dataset, focused on decision quality rather than raw accuracy.
- Engineered leakage-free features and validated them against the dataset's transaction-generation logic
- Designed cost-optimal decision thresholds derived from a business cost matrix instead of a default 0.5 cutoff
- Built per-prediction explainability using SHAP
- Implemented PSI/KS-based drift monitoring for production reliability
- Stack: XGBoost, FastAPI, PostgreSQL, Docker, Streamlit

[🔗 View Repository](https://github.com/S-Kamath-01/fraud-detection-cost-optimal)

---

## 🎬 Two-Tower Recommender System
Embedding-based recommendation engine built on MovieLens, designed for fast approximate nearest-neighbor retrieval.
- Implemented BPR (Bayesian Personalized Ranking) with rejection-sampled negatives
- Built temporal per-user train/test splits to avoid data leakage
- Designed dot-product scoring towers for FAISS-compatible retrieval
- Stack: PyTorch, FAISS, FastAPI, PostgreSQL, Streamlit

[🔗 View Repository](https://github.com/S-Kamath-01/two-tower-recys)

## 🚀 SpaceX Falcon 9 Landing Prediction
Machine learning project completed as part of the IBM Applied Data Science Capstone.
- Collected and processed launch data using APIs, SQL, and Python
- Built and evaluated classification models using Scikit-learn
- Compared Logistic Regression, SVM, KNN, and Decision Tree models
- Achieved **94.44% accuracy** using an optimized Decision Tree model
- Developed interactive dashboards and geospatial visualizations
